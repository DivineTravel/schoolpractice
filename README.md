try: 를 앞에 붙이고 끝에 except ValueError: 는 입력한 값이 적절하지 않을때 (숫자를 입력해야하는데 문자를 입력했다던가) 입력값이 적절하지 않다는 문구를 띄우기 위해 있는건데, 굳이 필요는 없어!
